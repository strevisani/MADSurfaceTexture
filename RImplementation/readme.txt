Folder content

"basicKernels.RData" The pre-computed kernels of order 1 and 2
"kernelsR" The folder with the pre-computed kernels in text format
"DATA" Some of the DEMs used in the paper: Trevisani, S. and Teza, G. and Guth, P., A Simplified Geostatistical Approach for Characterizing Key Aspects of Short-Range Roughness. Available at SSRN: https://ssrn.com/abstract=4223135 or http://dx.doi.org/10.2139/ssrn.4223135
"GeostTextureLibraryShared.R" The source code for performing geostatistical based roughness evaluation. It is also included roughness based on normal vectors dispersion
"scriptExplainingUsage.R" A simple R script showing how to use the main functions
